# 产品介绍

  • 国内稳定直连，无需魔法。任选空闲车辆，输入授权令牌直达，无需注册账号，快捷便利，用户无需承担封号风险。
  • 会话隔离，聊天记录随授权令牌保存在所使用的车辆内，不同令牌之间聊天记录均独立保存，保障用户隐私和数据安全。
  • 授权令牌每3小时50次GPT-4无限GPT-3.5，令牌次数标准随官网PLUS会员同步，支持使用GPT-4、Dall·E3文生图、联网、插件、上传文件、视觉识图以及各种GPTs等等功能，提供与官网相同的使用体验。

# 常见问题

  • 平台内普通账号和Plus账号有什么区别?
    • 普通账号只能使用3.5模型，PLUS账号可使用4.0、多模态、画图、识图等等、可使用GPTs应用以及插件功能。
  • 这个站点是什么模式?
    • 平台通过整合Plus账号进行二次调度高效利用，使用会话隔离保护隐私，非传统账密共享形式，拥有平台授权令牌即可进入任意空闲车队使用，高效调度，避免不可用情况。
  • 会话记录可以一直保存吗？
    • 因国外官网账号有被封号概率，所属车辆账号被封，后台替换账号后，该车辆内聊天记录会丢失，所以请务必本地手动备份重要聊天资料。
  • 可以使用别人创建的GPTs吗？
    • 当需要使用别人分享的GPTs时，把别人分享的GPTs链接里的chat.openai.com换成ShareGPT站点的网址即可。
