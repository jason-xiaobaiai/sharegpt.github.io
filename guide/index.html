# 常见问题

# 问：平台内普通账号和Plus账号有什么区别?

普通账号只能使用GPT-3.5模型，PLUS账号可使用GPT-4、GPT-4o、多模态、画图、识图、联网、数据分析、上传文件以及GPTs应用等功能。

# 问：授权账号是独享的吗？

是的，在线购买获得的授权账号都是独享的，可以使用全部Plus通道以及普通通道，别人看不到你的会话记录，同样你也看不到其他人的会话记录，不同账号之间聊天记录均为独立保存～

# 问：可以和别人拼团共用一个账号吗？

为了方便大家的使用，我们没有做多设备登录限制，大家可以和朋友拼团购买，共享同一个授权账号进行使用～

# 问：通道内的会话记录可以一直保存吗？

一般情况下，通道内的会话记录都是能够正常保存的，但因国外官方账号有被封号的概率，所属通道账号被封，后台替换账号后，该通道内聊天记录会丢失，所以还是请务必本地手动备份重要聊天资料～

# 问：可以使用别人创建的GPTs吗？

可以的，当需要使用别人创建的GPTs时，可以直接在GPTs商店里进行搜索，或者把别人分享的GPTs链接里的chat.openai.com换成ShareGPT站点的网址chat.nextai.today即可~

# 问：You've reached the current usage cap for GPT-4. 报错是什么意思？

这个是OpenAI官方的规定和限制，每个PLUS号目前都有40条/3小时的速率限制，根据我们的测试，工作日晚上（北京时间）Plus只有大约20次/3小时的GPT-4。如果你遇到You've reached the current usage cap for GPT-4.字样的报错无法使用，说明该PLUS通道速率已满，处于停运状态（需要等待一段时间才能恢复），只需要点击右上角“换车”更换空闲的通道尝试即可，在我们的直连通道内，3小时80次GPT-4大家是可以用满的，一个通道次数到限制了可以换通道继续用～
